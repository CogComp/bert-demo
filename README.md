### Running the demo 
To install the python dependencies, simply do: 
```python 
pip3.6 install -r requirements.txt
```

- Check if Django is installed:
 ```
 $ python -m django --version
 ```
 
 - Run the app: 
```
$ python3.6 manage.py runserver
```
